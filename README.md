# image-arrager
画像を横並びに配置できるデスクトップアプリをつくります

## Description

ライダーここに詳しい説明文を入れてください。

## Requirement

- npm

## Usage

1. 必要な node モジュールのインストール
  ```
  $ npm install
  ```

1. `npm start` を実行してファイルのElectronアプリを立ち上げる
  ```
  $ npm start
  ```

## Install
```
$ git clone git@github.com:canvas-ncu/image-arrager.git
```
